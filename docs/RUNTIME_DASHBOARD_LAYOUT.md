# Runtime Dashboard Layout

1. **Sovereignty State**: Operator key loaded, doctrine checksum, immutable mode status.
2. **Companion State**: Current tier, trait vector, bypass mode (must be false).
3. **Weaver Contract Airlock**: Last validation result, denied reasons, approval count.
4. **Event Bus Health**: Queue depth, throughput, backpressure state.
5. **Ledger & Vault**: Append-only write status, last flush timestamp.
6. **Agents Runtime**: Active agents, idle agents, error counts by taxonomy.
7. **Reality Sync**: Shadow queue size, ingest/drain rates.
8. **Dispatch Guardrail**: Dispatch accepted/rejected, override usage audit.
