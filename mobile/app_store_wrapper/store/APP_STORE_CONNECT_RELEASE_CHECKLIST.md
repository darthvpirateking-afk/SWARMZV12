# Apple App Store Release Checklist (SWARMZ)

## 1) App identity
- [ ] Final bundle identifier confirmed (example: `com.swarmz.app`)
- [ ] App name finalized: `SWARMZ`
- [ ] Version/build numbers set in Xcode

## 2) Legal + policy
- [ ] Privacy policy URL added in App Store Connect
- [ ] App Privacy questionnaire completed
- [ ] Age rating completed
- [ ] Export compliance answered

## 3) Assets
- [ ] App icon (1024x1024, no alpha)
- [ ] iPhone screenshots for required sizes
- [ ] Description, keywords, support URL, marketing URL

## 4) Build + signing
- [ ] Apple Developer signing/certs valid
- [ ] Archive built from Xcode
- [ ] Archive uploaded via Organizer
- [ ] TestFlight sanity test completed

## 5) Release
- [ ] App Review notes added (pairing flow, account details if needed)
- [ ] Submit for review
- [ ] Post-release crash monitoring enabled
