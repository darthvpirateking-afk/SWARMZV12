# Android Wrapper for NEXUSMON

## Overview

This Android wrapper uses a WebView to load the NEXUSMON dashboard.

## Features

- Loads the NEXUSMON dashboard from `http://127.0.0.1:8012` or a LAN URL.
- Simple and lightweight.

## Setup

1. Place `MainActivity.kt` in your Android project.
2. Ensure the app has permissions to access the network.
3. Build and run the app to view the NEXUSMON dashboard.
