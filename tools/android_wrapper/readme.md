# Android Wrapper for SWARMZ

## Overview

This Android wrapper uses a WebView to load the SWARMZ dashboard.

## Features

- Loads the SWARMZ dashboard from `http://127.0.0.1:8012` or a LAN URL.
- Simple and lightweight.

## Setup

1. Place `MainActivity.kt` in your Android project.
2. Ensure the app has permissions to access the network.
3. Build and run the app to view the SWARMZ dashboard.
