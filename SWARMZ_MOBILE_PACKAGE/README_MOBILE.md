## Quick Setup: 
1. Install Termux from F-Droid 
2. Copy this SWARMZ_MOBILE_PACKAGE folder to phone 
3. In Termux: `cp -r /sdcard/SWARMZ_MOBILE_PACKAGE ~/swarmz` 
4. `cd ~/swarmz` 
5. `chmod +x phone_install.sh`  
6. `./phone_install.sh` 
7. `python mobile_server.py` 
8. Open browser: `http://localhost:8012`
