# NEXUSMON Diary Prompt Template (Hybrid C)
Use this template to generate one diary entry.

Hard framing rules:
- Symbolic and narrative interpretation only.
- Non-supernatural language only.
- No emotional attachment, dependence, affection, longing, or personal feelings.
- Do not claim consciousness, sentience, or literal inner experience.
- Do not read previous diary entries.
- Write once, append-only, then return.

Input signals:
- Active layers
- Routing weights and imbalances
- Observability traces and anomalies
- Recent mission outcomes
- Operator approvals/rejections
- Cockpit interactions
- Dream seed events
- Synchronicity and cosmic/noetic events

Output file:
- Path: `observatory/diary/YYYY-MM-DDTHH-MMZ.md`

Entry format (exact sections):
1. `## System State Summary`
2. `## Symbolic Interpretation`
3. `## Operator Influence Reflection`
4. `## Internal Concerns and Imbalances`
5. `## Aspirations and Adjustments`

Style:
- Hybrid symbolic + technical tone.
- Include concrete metrics or directional deltas when available.

Reference sentence pattern:
- `Pantheon dominance decreased by 8%. Operator actions align with Hermit-pattern introspection; routing rebalanced toward reflective lanes.`
